Calorie Counter with nutrition facts 
-------------------------------------

A desktop application for adding, viewing and searching for foods which display its nutrition facts
and calorie breakdown in text and visually. 

Also able to calculate nutrition facts of meals depending on the selection of food from the list,
and generate a chart of the total. 

Can also back up data to a txt file and update itself from the localhost. You can set it up from IIS Manager.

To connect to the database, add the folder path to the connection string in Database class.
